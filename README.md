# 🍔 The Food You Love

A modern, responsive landing page to help users find delicious food near them. Built with plain HTML, CSS, and JavaScript, featuring a full-width hero banner, postcode search form, product cards grid, and a scroll-to-top button.

## 🔧 Tech Stack

- **HTML5** — Semantic markup  
- **CSS3** — Flexbox & Grid, custom properties for theming  
- **JavaScript (ES6)** — DOM interactions, smooth scrolling, form handling  

---

## ✨ Features

- **Hero Banner** with dark overlay & centered call-to-action  
- **Postcode Search Form**  
  - Validates non-empty input  
  - Alerts/logs the entered postcode  
- **Responsive Products Grid**  
  - Three food item cards with image, heading, and description  
  - Hover lift effect on each card  
- **Scroll-to-Top Button**  
  - Appears after scrolling down  
  - Smooth scroll back to top
